## 🎨 Design Update

**Versi ini menggunakan:**
- Bright Vault color scheme (Cyan-Purple gradient)
- Team Layerwise logo di header
- Colorful modern design untuk presentasi
- Enhanced visual effects & animations

---

## ✨ Fitur Utama

- **🔍 Prediksi Risiko Banjir Real-time** menggunakan algoritma layerwise
- **📊 Analisis 4 Layer** dengan bobot berbeda (Rain 40%, Humidity 25%, Clouds 20%, Wind 15%)
- **📈 Visualisasi Data** dengan grafik prakiraan 24 jam
- **⚡ Update Real-time** jam WIB/UTC
- **📱 Responsive Design** colorful & modern
- **🌐 Ready for API** OpenWeather OneCall 3.0

---

## 🛠️ Tech Stack

- **React 18** - UI framework
- **Vite 5.x** - Build tool
- **Tailwind CSS 3.x** - Styling
- **Lucide React** - Icons
- **OpenWeather API 3.0** - Data source (planned)

## 📦 Instalasi

### Prerequisites
```bash
Node.js v18+
npm atau yarn
```

### Install Dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build Production
```bash
npm run build
```
